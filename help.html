Need help?

If you need help with Docusaurus, you can try one of the mechanisms below.

Browse the docs

Learn more about Docusaurus using the official documentation.

Discord

You can join the conversation on Discord on one of our two text channels: #docusaurus-users for user help and #docusaurus-dev for contributing help.

Twitter

You can follow and contact us on Twitter.

GitHub

At our GitHub repo Browse and submit issues or pull requests for bugs you find or any new features you may want implemented. Be sure to also check out our contributing information.
